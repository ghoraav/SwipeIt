# SwipeIt
The fast, fun way to clean your photo gallery.

# HOW TO DOWNLOAD:
1) Scroll down, locate releases section
2) Click "version 1 (beta tesing)"
3) Scroll down
4) Locate the apk file "SwipeIt.apk"
5) Install the apk. (During installation, Google Play Protect may warn that the app is "Unsafe." To proceed, tap More details on the warning screen, and then select Install anyway.)
